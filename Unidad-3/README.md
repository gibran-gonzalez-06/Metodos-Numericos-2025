# Unidad 3 - Solución de Sistemas de Ecuaciones

## Descripción

En esta unidad se estudiaron métodos iterativos para resolver sistemas de ecuaciones lineales. Se analizaron los procedimientos de cálculo y convergencia mediante la aplicación de los métodos de Jacobi y Gauss-Seidel.

## Objetivo

Resolver sistemas de ecuaciones lineales utilizando métodos numéricos iterativos y comparar la eficiencia de cada procedimiento.

## Actividades realizadas

- Resolución de ejercicios propuestos en clase.
- Aplicación de los métodos de Jacobi y Gauss-Seidel.
- Verificación de resultados mediante iteraciones sucesivas.
- Desarrollo de programas en Java para automatizar los cálculos.

## Programas desarrollados

### Jacobi.java

Programa que implementa el método de Jacobi para resolver sistemas de ecuaciones lineales mediante iteraciones.

### GaussSeidel.java

Programa que implementa el método de Gauss-Seidel para obtener soluciones aproximadas de sistemas de ecuaciones lineales.
