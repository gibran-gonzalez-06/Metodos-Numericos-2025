# Evidencias de Ejecución - Unidad 3

## Descripción

Esta carpeta contiene capturas de pantalla de la ejecución de los programas desarrollados en Java.

## Evidencias incluidas

### Método de Jacobi

Las capturas muestran:

- Ingreso del número de ecuaciones.
- Captura de coeficientes del sistema.
- Ingreso de valores iniciales.
- Tolerancia utilizada.
- Iteraciones realizadas.
- Error mayor calculado.
- Resultado final aproximado.

### Método de Gauss-Seidel

Las capturas muestran:

- Ingreso del sistema de ecuaciones.
- Valores iniciales.
- Tolerancia y número máximo de iteraciones.
- Resultados obtenidos en cada iteración.
- Solución aproximada final.

## Objetivo

Comprobar que los programas se ejecutan correctamente y generan aproximaciones válidas para los sistemas de ecuaciones.
