# Evidencias de Ejecución - Unidad 4

## Descripción

Esta carpeta contiene las capturas de pantalla que evidencian la correcta ejecución de los programas desarrollados en Java para la Unidad 4.

## Evidencias incluidas

### Método del Trapecio

Se muestra la ejecución del programa solicitando:

- Límite inferior (a)
- Límite superior (b)
- Número de intervalos

Posteriormente se calcula el valor aproximado de la integral mediante la Regla del Trapecio.

### Método de Simpson 1/3

Se muestra la ejecución del programa solicitando:

- Límite inferior
- Límite superior
- Número par de intervalos

Finalmente se calcula el valor aproximado de la integral utilizando el Método de Simpson 1/3.

## Objetivo

Verificar el correcto funcionamiento de los algoritmos implementados y documentar los resultados obtenidos durante su ejecución.
