# Unidad 4 - Integración Numérica

## Descripción

En esta unidad se estudiaron métodos de integración numérica utilizados para aproximar el valor de integrales definidas cuando su solución analítica resulta complicada o imposible de obtener.

## Objetivo

Aplicar métodos numéricos de integración para aproximar áreas bajo la curva mediante algoritmos computacionales.

## Actividades realizadas

- Resolución de ejercicios propuestos en clase.
- Aplicación de la Regla del Trapecio.
- Aplicación de la Regla de Simpson.
- Comparación de resultados obtenidos.
- Desarrollo de programas en Java para automatizar los cálculos.

## Programas desarrollados

### ReglaTrapecio.java

Programa que calcula aproximaciones de integrales definidas mediante la Regla del Trapecio.

### ReglaSimpson.java

Programa que calcula aproximaciones de integrales definidas mediante la Regla de Simpson.
