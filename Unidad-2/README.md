# Unidad 2 - Métodos de Solución de Ecuaciones

## Instituto Tecnológico Superior de Huauchinango

### Carrera

Ingeniería en Sistemas Computacionales

## Descripción

En esta unidad se estudiaron métodos numéricos para encontrar raíces de ecuaciones no lineales mediante procedimientos iterativos.

Se analizaron métodos cerrados y abiertos para aproximar soluciones de ecuaciones matemáticas utilizando programación en Java y actividades desarrolladas durante el curso.

## Actividades realizadas

- Resolución de ejercicios en Microsoft Excel.
- Aplicación del Método de Bisección.
- Aplicación del Método de Newton-Raphson.
- Desarrollo de programas en Java para automatizar los cálculos.
- Análisis de iteraciones y errores obtenidos.

## Programas desarrollados

- MetodoBiseccion.java
- NewtonRaphson.java

## Temas trabajados

- Método de Bisección.
- Método de Newton-Raphson.
- Aproximación de raíces.
- Error numérico.
- Iteraciones sucesivas.
