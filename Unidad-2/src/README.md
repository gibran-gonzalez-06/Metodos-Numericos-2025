# Método de Newton-Raphson

## Descripción

Este programa implementa el Método de Newton-Raphson para aproximar raíces de ecuaciones no lineales.

La función utilizada es:

f(x) = x² - 2

y su derivada:

f'(x) = 2x

El programa realiza iteraciones sucesivas hasta encontrar una aproximación suficientemente cercana a la raíz real.

## Funcionalidades

- Cálculo de f(x).
- Cálculo de f'(x).
- Aplicación de la fórmula de Newton-Raphson.
- Visualización de cada iteración.
- Cálculo del error en cada paso.
- Obtención de la raíz aproximada.

## Resultado esperado

La raíz aproximada obtenida es:

√2 ≈ 1.41421356

## Lenguaje utilizado
