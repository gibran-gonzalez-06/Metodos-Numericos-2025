# Unidad 6 - Ecuaciones Diferenciales Ordinarias

## Descripción

En esta unidad se estudiaron métodos numéricos para la solución aproximada de ecuaciones diferenciales ordinarias (EDO). Se realizaron ejercicios prácticos utilizando diferentes técnicas de aproximación.

## Objetivo

Aplicar métodos numéricos para obtener soluciones aproximadas de ecuaciones diferenciales ordinarias y sistemas de ecuaciones diferenciales.

## Actividades realizadas

- Resolución de ejercicios del cuadernillo.
- Método de Euler.
- Método de Euler Mejorado (Heun).
- Método de Runge-Kutta de Cuarto Orden (RK4).
- Solución de sistemas de ecuaciones diferenciales.
- Desarrollo de procedimientos paso a paso para cada ejercicio.

## Evidencias

En esta unidad no se desarrolló un programa computacional. El trabajo realizado consistió en la resolución y análisis de ejercicios propuestos en el cuadernillo de la materia.
