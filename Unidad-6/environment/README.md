# Entorno de Trabajo - Unidad 6

## Descripción

Esta carpeta documenta el entorno utilizado para el desarrollo de las actividades correspondientes a la Unidad 6.

## Recursos utilizados

- Libreta de apuntes.
- Material proporcionado por el docente.
- Calculadora científica.
- Apuntes de clase.
- Presentaciones de apoyo.

## Metodología de trabajo

Las actividades fueron desarrolladas de forma manual mediante la resolución de ejercicios y problemas relacionados con ecuaciones diferenciales ordinarias, siguiendo los procedimientos explicados durante las sesiones de clase.

## Observaciones

Durante esta unidad no fue necesario desarrollar programas de software, ya que el enfoque principal estuvo orientado a la comprensión y aplicación manual de los métodos numéricos estudiados.

## Objetivo

Registrar las condiciones y recursos utilizados para la realización de las actividades académicas de la unidad.
