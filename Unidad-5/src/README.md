# Código Fuente - Unidad 5

## Descripción

Esta carpeta contiene el programa desarrollado en Java para aplicar el Método de Interpolación de Lagrange.

## Archivo incluido

### MetodoLagrange.java

Programa que permite calcular un valor interpolado a partir de un conjunto de puntos conocidos.

El usuario ingresa:

- Cantidad de puntos.
- Valores de x.
- Valores de y.
- Valor de x que se desea interpolar.

Posteriormente, el programa calcula los términos del polinomio de Lagrange y muestra el procedimiento realizado.

## Ejemplo ejecutado

Puntos utilizados:

- (1, 1)
- (2, 4)
- (3, 9)

Valor a interpolar:

- x = 2.5

Resultado obtenido:

- P(2.5) = 6.25

## Objetivo

Automatizar el cálculo de interpolación mediante el Método de Lagrange utilizando programación en Java.

## Lenguaje utilizado

Java
