# Entorno de Ejecución - Unidad 5

## Descripción

Esta carpeta documenta el entorno utilizado para desarrollar y ejecutar el programa correspondiente a la Unidad 5.

## Software utilizado

- Java
- Visual Studio Code
- JDK 17 o superior
- Sistema operativo Windows

## Librerías utilizadas

No se utilizaron librerías externas.

El programa fue desarrollado únicamente con Java estándar.

## Ejecución

1. Abrir el archivo MetodoLagrange.java.
2. Compilar el programa.
3. Ejecutar el programa.
4. Ingresar la cantidad de puntos.
5. Capturar los valores de x y y.
6. Ingresar el valor que se desea interpolar.
7. Revisar el procedimiento y resultado final.

- MetodoLagrange.java
