# Documentación - Unidad 5

## Descripción

Esta carpeta contiene las actividades y ejercicios realizados durante la Unidad 5.

## Contenido

- Evidencias de ejercicios realizados en libreta.
- Procedimientos relacionados con interpolación.
- Ejercicios del cuadernillo.
- Actividades académicas de la unidad.

## Temas incluidos

- Interpolación de Lagrange.
- Diferencias divididas de Newton.
- Mínimos cuadrados.
- Correlación.

## Objetivo

Documentar los ejercicios y procedimientos realizados durante la unidad para respaldar el aprendizaje de los métodos de interpolación y ajuste de funciones.
