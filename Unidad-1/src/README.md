# Programa de Errores Numéricos

## Descripción

Este programa fue desarrollado en Python para aplicar conceptos básicos de errores numéricos utilizados en Métodos Numéricos.

El programa permite realizar diferentes cálculos relacionados con la precisión de los datos y la propagación de errores.

## Funciones implementadas

### 1. Error Absoluto y Relativo

Calcula:

- Error absoluto
- Error relativo
- Error relativo porcentual

A partir de un valor real y un valor aproximado proporcionados por el usuario.

### 2. Cifras Significativas

Determina la cantidad de cifras significativas que contiene un número ingresado por el usuario.

### 3. Propagación de Error en una Suma

Calcula el error total resultante al sumar dos cantidades con errores asociados.

### 4. Propagación de Error en una Multiplicación

Calcula el error propagado cuando se realiza una multiplicación entre dos valores que poseen incertidumbre.

## Objetivo

Comprender la importancia de los errores numéricos y su influencia en los resultados obtenidos mediante métodos computacionales.
