# Unidad 1 - Introducción a Métodos Numéricos

## Objetivo

Comprender los conceptos básicos de los métodos numéricos y su aplicación en la solución de problemas matemáticos mediante herramientas computacionales.

## Actividades realizadas

- Resolución de ejercicios propuestos en clase.
- Desarrollo de programa en Python.
- Elaboración de actividades en Microsoft Excel.
- Análisis de resultados obtenidos.

## Archivos incluidos

### src

Contiene el código fuente desarrollado para la unidad.

### docs

Espacio destinado para documentación, reportes o evidencias.

### environment

Información del entorno de desarrollo utilizado.

### tests

Pruebas y ejercicios de validación.

- Python
