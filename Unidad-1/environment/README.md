# Entorno de Ejecución

## Requisitos

- Python 3.10 o superior
- Visual Studio Code

## Librerías utilizadas

El programa utiliza únicamente librerías estándar de Python.

No requiere instalación de dependencias adicionales.

## Ejecución

Ejecutar el archivo:

ejercicio_errores.py

desde Visual Studio Code o desde la terminal de Python.
